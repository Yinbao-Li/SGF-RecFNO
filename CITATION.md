# Citation

## SGF-RecFNO / IsoRecFNO (Yinbao Li)

If you use **SGF-RecFNO** or **IsoRecFNO** from this repository, please cite:

```bibtex
@misc{li2026sgfrecfno,
  author = {Li, Yinbao},
  title = {SGF-RecFNO: Self-Geometry Feedback RecFNO for Sparse Heat Field Reconstruction},
  year = {2026},
  publisher = {GitHub},
  url = {https://github.com/Yinbao-Li/SGF-RecFNO}
}
```

> Once a formal paper is published, replace this `@misc` entry with the journal or conference BibTeX.

## RecFNO (required)

This codebase builds on RecFNO. Please cite the original work as well:

```bibtex
@misc{zhao2023recfno,
  doi = {10.48550/ARXIV.2302.09808},
```
